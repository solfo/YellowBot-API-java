YellowBot API Java Client
=========================

Welcome to the YellowBot API Java client project.

Building Instructions
---------------------

Running the command under the root directory of the project

    ant -f yellowbot-api.xml build.dist

should generate under "out/dist/" a distribution file

    yellowbot-api-0.1.0.zip

containing

    examples/
    lib/
    javadoc/
